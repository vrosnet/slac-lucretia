The LUCRETIA Project

Downloads

The Lucretia code is entirely open-source:  anyone can freely download any or all of the source, examples, binaries, etc.  Links to facilitate downloading are below.  This is also the place to look for downloads of the mex-ed XSIF parsing tools.

Lucretia Downloads
The complete Lucretia directory tree (source files, mexfile binaries, examples, m-files) is available as a Zip'ed archive.  The archive is available in WinXP or Unix flavor (the difference being the end-of-line termination of the flat text files; otherwise the contents of the 2 Zip files are the same).  Alternately the file tree can be browsed here.

The Lucretia website web pages are available as a Zip'ed archive as well.  Download the archive here.

XSIF

The XSIF source is available in zip'ed form for WinXP or unix/Linux, and the source can be browsed here.  The dll/mexsol/mexglx files are available in zip'ed form here.
